Personvern og informasjonskapsler

ReNew Detailing Studio (enkeltpersonforetak, Bergen) tar personvernet ditt på alvor. Denne siden forklarer kort hvordan vi bruker informasjonskapsler (cookies) på nettsiden.

Informasjonskapsler

Nødvendige informasjonskapsler kreves for at nettsiden skal fungere, og lagrer blant annet valget ditt om samtykke. Disse er alltid på.

Statistikk og markedsføring (for eksempel Google og Meta) brukes kun hvis du samtykker. Ingen slike informasjonskapsler settes før du har trykket «Godta alle». Frem til vi starter markedsføring, samler vi ikke inn slike data.

Du kan når som helst endre valget ditt via lenken Informasjonskapsler nederst på siden.
